# DDoS-TK
DDoS Tool, with TCP and HTTP. Using Argparse CLI. (DOS, not DDoS)

# Install
git clone https://github.com/TKMASTERPYBAT/DDoS-TK.git



cd DDoS-TK



chmod +x setup.sh



sudo ./setup.sh

# Use
Example:


  DDOS -t 127.0.0.1 -p 22 -m TCP -a 100


  DDOS -t https://example.com -m HTTP -a 100
